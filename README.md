# Python-learning-journey
Mi camino a ser experto en Python

This repository documents my Python learning path as part of my roadmap toward Data Analyst / BI Analyst roles, with future growth into data engineering, data science, and applied AI.

The goal of this repository is to practice Python through small exercises, CLI programs, and object-oriented programming projects. Some exercises are for learning purposes, while others may evolve into portfolio-ready projects.

## Repository Structure

```text
py_01_python_core/
```

Contains Python fundamentals and practice exercises, including:

* Variables and data types
* Conditionals
* Loops
* Functions
* Lists and dictionaries
* Error handling with try/except
* CLI programs such as the Task Manager

```text
poo_01_oop_projects/
```

Contains object-oriented programming projects and exercises, including:

* Classes and objects
* Inheritance
* Methods and attributes
* Basic software structure
* OOP mini projects such as a library management system
